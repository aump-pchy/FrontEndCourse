// 01 — what is html

HTML คืออะไร?

HTML ย่อมาจาก HyperText Markup Language
เป็นภาษาที่ใช้ กำหนดโครงสร้าง ของหน้าเว็บ

ทุกเว็บไซต์บนโลกนี้ สร้างจาก HTML ทั้งนั้น

HTML โครงสร้าง / กระดูก
CSS สไตล์ / สีหน้าตา
JavaScript พฤติกรรม / ลูกเล่น
💡 Tim Berners-Lee สร้าง HTML ในปี 1991
เพื่อให้นักวิทยาศาสตร์แชร์เอกสารกัน — นั่นแหละคือจุดกำเนิดของ World Wide Web
รูปแบบพื้นฐานของ HTML Tag:
<!-- HTML Tag มีรูปแบบนี้ --> <tagname> เนื้อหา </tagname> <!-- Opening tag Closing tag --> <h1>ยินดีต้อนรับ</h1> <p>นี่คือย่อหน้า</p> <!-- Tag ที่ปิดตัวเอง (Self-closing) --> <br> <!-- ขึ้นบรรทัดใหม่ --> <img src="cat.jpg">
นามสกุลไฟล์:  .html  หรือ  .htm
เปิดด้วย VSCode เขียน → บันทึก → เปิดใน Browser ดูผล
// 02 — html structure

โครงสร้างไฟล์ HTML

<!DOCTYPE html> <html lang="th"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width"> <title>ชื่อหน้าเว็บ</title> </head> <body> <!-- เนื้อหาที่แสดงผลในเว็บ --> <h1>สวัสดีทุกคน!</h1> <p>ยินดีต้อนรับ</p> </body> </html>
ความหมายของแต่ละส่วน:
<!DOCTYPE html>
บอก browser ว่านี่คือ HTML5
<html lang="th">
root element — ครอบทุกอย่าง, lang บอกภาษา
<head>
ข้อมูล metadata — ไม่แสดงบนหน้าเว็บ
<meta charset="UTF-8">
รองรับภาษาไทยและ Unicode
<title>
ชื่อที่แสดงบน Tab ของ browser
<body>
เนื้อหาที่ผู้ใช้มองเห็น — ทุกอย่างอยู่ที่นี่
🖊️ Indentation สำคัญ!
เว้นวรรค (indent) ให้เห็นโครงสร้าง — ทำให้โค้ดอ่านง่าย ไม่ใช่แค่ความสวยงาม
// 03 — headings

Heading Tags h1 → h6

<h1>หัวข้อใหญ่สุด (h1)</h1> <h2>หัวข้อรอง (h2)</h2> <h3>หัวข้อย่อย (h3)</h3> <h4>หัวข้อ (h4)</h4> <h5>หัวข้อเล็ก (h5)</h5> <h6>หัวข้อเล็กสุด (h6)</h6>
📌 กฎสำคัญ: ในหน้าหนึ่งควรมี <h1> แค่ 1 ตัว — Google ใช้มันระบุว่าหน้านี้พูดถึงอะไร
ตัวอย่างผลลัพธ์ใน Browser:
h1 — หัวข้อใหญ่สุด
h2 — หัวข้อรอง
h3 — หัวข้อย่อย
h4 — หัวข้อ
h5 — หัวข้อเล็ก
h6 — หัวข้อเล็กสุด
// 04 — paragraph & text

Paragraph & Text Tags

<!-- ย่อหน้าข้อความ --> <p>ข้อความในย่อหน้า...</p> <!-- เน้นความสำคัญ --> <strong>ข้อความหนา/สำคัญ</strong> <!-- เน้นน้ำเสียง --> <em>ข้อความเอียง</em> <!-- ขึ้นบรรทัดใหม่ --> ข้อความ<br>บรรทัดใหม่ <!-- เส้นแบ่งแนวนอน --> <hr>
ตัวอย่างผลลัพธ์:

ข้อความในย่อหน้าแรก นี่คือตัวอย่างประโยค

ใช้ ตัวหนา สำหรับสิ่งสำคัญ และ ตัวเอียง สำหรับคำศัพท์

ข้อความหลังเส้นแบ่ง

💡 <br> และ <hr> คือ Self-closing tag — ไม่มี tag ปิด ไม่ต้องเขียน </br>
// 05 — lists

ul & ol — List Tags

🔵 Unordered List — ไม่มีลำดับ:
<ul> <li>ข้าวผัดกุ้ง</li> <li>ผัดกะเพราหมู</li> <li>ต้มยำกุ้ง</li> </ul>
🟠 Ordered List — มีลำดับ:
<ol> <li>เปิด VSCode</li> <li>สร้างไฟล์ index.html</li> <li>บันทึกและเปิด Browser</li> </ol>
ตัวอย่างผลลัพธ์:
Unordered
  • ข้าวผัดกุ้ง
  • ผัดกะเพราหมู
  • ต้มยำกุ้ง
Ordered
  1. เปิด VSCode
  2. สร้างไฟล์
  3. เปิด Browser
📌 ul = ใช้กับรายการที่ ลำดับไม่สำคัญ (เมนู, งานอดิเรก)
📌 ol = ใช้กับรายการที่ ลำดับสำคัญ (ขั้นตอน, อันดับ)
// 06 — hyperlinks

Link Tag <a href="...">

<!-- ลิงก์ไปเว็บอื่น --> <a href="https://google.com"> ไปยัง Google </a> <!-- เปิดใน Tab ใหม่ --> <a href="https://google.com" target="_blank"> เปิด Tab ใหม่ </a> <!-- ลิงก์ไฟล์ในโปรเจกต์ --> <a href="menu.html">ดูเมนู</a> <!-- ลิงก์ส่งอีเมล --> <a href="mailto:me@mail.com"> ติดต่อเรา </a>
Attributes ที่ใช้บ่อย:
href="..."
URL ปลายทาง — ใส่ address เต็ม หรือชื่อไฟล์
target="_blank"
เปิดลิงก์ใน Tab ใหม่ ไม่แทนที่หน้าปัจจุบัน
mailto: / tel:
ลิงก์พิเศษ — เปิด Email App / โทรออก
💡 ถ้า href เป็นชื่อไฟล์ เช่น "menu.html" — ไฟล์นั้นต้องอยู่ใน folder เดียวกัน
// 07 — images

Image Tag <img>

<!-- รูปจากไฟล์ --> <img src="images/profile.jpg" alt="รูปโปรไฟล์ของฉัน" width="200" > <!-- รูปจาก URL --> <img src="https://example.com/img.jpg" alt="คำอธิบายรูปภาพ" > <!-- กำหนดทั้งกว้างและสูง --> <img src="food.jpg" alt="เมนูอาหาร" width="300" height="200">
Attributes สำคัญ:
src="..."
ที่อยู่ไฟล์รูป — ชื่อไฟล์หรือ URL เต็ม
alt="..." ⚠️ บังคับ
ข้อความแทนรูป — ช่วย Screen Reader & SEO
width / height
ขนาด (px) — ใส่แค่อย่างเดียว scale อัตโนมัติ
ไฟล์รูปที่รองรับ: .jpg .png .gif .svg .webp
แนะนำใส่รูปไว้ใน folder images/ เพื่อความเป็นระเบียบ
// workshop — ชิ้นงาน week 01

ชิ้นงาน: แนะนำตัวเอง + เมนูร้านอาหาร

📋 สิ่งที่ต้องทำ
  • 1
    index.html — แนะนำตัวเอง
    h1-h3 ชื่อและหัวข้อ, p ข้อมูลส่วนตัว, ul งานอดิเรก, img รูปตัวเอง, a ลิงก์ไปหน้าเมนู
  • 2
    menu.html — เมนูร้านอาหาร
    h1 ชื่อร้าน, h2 หมวดหมู่, ol/ul รายการเมนู, img รูปอาหาร, a ลิงก์กลับหน้าแรก
  • 3
    เชื่อมสองหน้า
    มีลิงก์ไปมาระหว่างสองหน้า ใช้ <a href>
โครงสร้างไฟล์:
📁 my-website/ │ ├── index.html ← หน้าแนะนำตัว ├── menu.html ← หน้าเมนูอาหาร └── images/ ├── profile.jpg ← รูปตัวเอง └── food.jpg ← รูปอาหาร
เกณฑ์คะแนน:
✓ ใช้ Tag ครบทุกประเภท (h, p, ul/ol, a, img)
✓ มี alt attribute ทุกรูป
✓ HTML ถูกต้อง — ปิด Tag ครบ
✓ มีลิงก์เชื่อมระหว่างสองหน้า
