Responsive Design

ทำให้เว็บแสดงผลดีบนทุกขนาดหน้าจอ — มือถือ แท็บเล็ต และ Desktop

📱 Mobile-First คืออะไร?

Tailwind ใช้แนวคิด Mobile-First —
เขียน CSS สำหรับมือถือก่อน แล้วค่อย override เมื่อหน้าจอใหญ่ขึ้น

/* ไม่มี prefix = มือถือ (ทุกขนาด) */ <div class="grid-cols-1"> /* md: = เมื่อหน้าจอ ≥ 768px */ <div class="grid-cols-1 md:grid-cols-2"> /* lg: = เมื่อหน้าจอ ≥ 1024px */ <div class="grid-cols-1 md:grid-cols-2 lg:grid-cols-3">

👁 ซ่อน-แสดง Element ตามขนาดหน้าจอ
<!-- ซ่อนบนมือถือ แสดงบน md+ --> <div class="hidden md:flex gap-6"> <a href="#">Home</a> <a href="#">About</a> </div> <!-- แสดงบนมือถือ ซ่อนบน md+ --> <button class="md:hidden"> ☰ เมนู </button> <!-- text ขนาดต่างกันตามหน้าจอ --> <h1 class="text-2xl md:text-3xl lg:text-5xl"> หัวข้อใหญ่ </h1>
hidden md:flex
ซ่อน (display:none) บนมือถือ, แสดงเป็น flex เมื่อ md+
md:hidden
แสดงตามปกติบนมือถือ, ซ่อนเมื่อ md+
text-sm md:text-base lg:text-lg
ขนาด text เปลี่ยนตาม breakpoint
