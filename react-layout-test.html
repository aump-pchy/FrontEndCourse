🚀 ขั้นตอนการทำ
1
สร้าง React Project (ทำ 2 Project แยกกัน หรือ Project เดียวก็ได้)
npm create vite@latest layout-a -- --template react cd layout-a && npm install
2
ติดตั้ง Tailwind CSS v4
npm install tailwindcss @tailwindcss/vite
เพิ่ม tailwindcss() ใน vite.config.js และ @import "tailwindcss" ใน index.css
3
สร้าง Components ใน src/components/
// ตัวอย่าง Component พื้นฐาน const SiteHeader = () => { return ( <header className="text-center py-6 bg-white border-b"> <h1 className="text-2xl font-bold">Company Name</h1> <p className="text-gray-500 text-sm">Tagline here</p> </header> ) } export default SiteHeader
4
ประกอบ Components ใน App.jsx
import SiteHeader from './components/SiteHeader' import NavBar from './components/NavBar' // ... import อื่นๆ const App = () => { return ( <div> <SiteHeader /> <NavBar /> <div className="flex gap-6 p-6"> {/* components ที่เหลือ */} </div> </div> ) } export default App
5
Push GitHub
git add . git commit -m "React Layout Test — Layout A & B" git push

📐 Week 4 — Grid & Responsive — 25 คะแนน

Responsive Layout B 2-col ใช้ flex-col md:flex-row / เมนูใช้ hidden md:flex ถ้ามี 10
✨ Tailwind + คุณภาพโค้ด — 15 คะแนน

โค้ดสะอาด อ่านง่าย ชื่อ Component สื่อความหมาย, มี export default ครบ, import ถูกต้อง 7
